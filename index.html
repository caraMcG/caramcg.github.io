Sorting

A Java program that uses bubble, selection, insertion and merge sorts. This was implemented using an ADT that represents a polygon and uses a compareTo method. The user enters a command line argument, the program then reads the text file to sort and print the first and every thousandth value of an array.

  • Java
  • ADT

Sorting Algorithms

Here you can see the set up of the objects which all extend the abstract data type 'Polygon'.

The arguments entered into the command line are loaded into the main and passed to 'PolygonSorterForName' to be processed. The results then are loaded into an array and sorted based on the argument sort chosen (bubble,insertion,merge, etc...)

An example of the output can be seen here from the command line. The arguments include 'F - file name', 'T - compare type' and 'S - chosen Sort'.

Node.js API

A Node.js project that utilizes axios, express and cheerio packages. This API scrapes adoptable dog data from shelters in and around the Calgary area; AARCS, PawsitiveMatch and Calgary Humane Society. GET calls return a JSON object that holds a URL, Name, Picture and ID (count) of the dog object.

  • Node.js
  • Axios
  • Express
  • Cheerio / JQuery

Calgary Shelter API

First step was to intialize the 3 packages used in this project.

Axios: Performing HttpRequests from node.js
Cheerio: Parses through markup language and provides an API for manipulating/traversing
Express: Routing used to handle GET requests

Second step was to set up the array of objects I would be gathering the data from. Each Shelter object is set up with a name and an address

Almost done!
I looped through each shelter and made an axios get request using each shelters address property. The .then method made it possible for the response data (html) to be loaded by Cheerio, which allowed for easy parsing using JQuery.

Lastly! The resulting data can then be viewed on my localhost via the /dogs endpoint.
